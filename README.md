# Tesco-DCF-Valuation
This project is an independently built discounted cash flow (DCF) valuation model of Tesco plc, developed to strengthen my financial modelling and valuation skills.

This model includes:

Unlevered DCF valuation

Forecast period: 2026–2030

WACC-based discounting

Sensitivity analysis table


Key Outputs:

Enterprise Value £47,432M

Equity Value £35,017M

Implied Share Price £5.20
